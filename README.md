# Automation Testing – Logout Functionality

This project demonstrates a simple logout functionality test developed in Java.

The test simulates a logged-in user, performs a logout action, and verifies that the user session is successfully terminated.

## Test Case

**Test Case ID:** TC_LOGOUT_01

**Feature Tested:** Logout Functionality

**Expected Result:** User should be logged out successfully.

**Status:** PASS
